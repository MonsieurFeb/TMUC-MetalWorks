# Installation
- Установите Minecraft 1.10.2. Выберите одну из папок (пустых) и запомните её.
- Установите Forge 12.18.3.2511, выбрав ту же папку.
- Переместите файлы из архива в папку игры
- Некоторые моды я не могу разместить здесь, поэтому их придётся дозагрузить самостоятельно
    - К ним относится оптифайн, по [этой](https://optifine.net/copyright) причине, скачать можно [здесь](http://optifine.net/adloadx?f=OptiFine_1.10_HD_U_I5.jar)
- Готово. Используемые мной настройки игры и модов также включены.
